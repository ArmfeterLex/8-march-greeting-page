# 8 March Greeting Page

Интерактивная веб-страница для поздравления с 8 Марта на чистом HTML, CSS и JavaScript.

## Функциональность

### 🎨 Интерактивные элементы
- **Заголовок** — меняет цвет при клике, временно изменяет фон страницы
- **Цветок** — меняет цвет головки при клике (случайный выбор из 7 цветов)
- **Дата** — автоматически отображает текущую дату

### 📝 Персональное поздравление
- Форма с вводом имени
- Проверка на пустое поле перед отправкой
- Персонализированное обращение с подстановкой имени

### 🎞️ Автоматический слайдер
- Запускается после отправки имени
- Три праздничных изображения сменяют друг друга
- Интервал смены 2 секунды
- Автоматическая остановка при закрытии открытки

### 🎁 Открытка
- Персональное приветствие
- Поздравительный текст
- Кнопка закрытия с остановкой слайдера

## Важное примечание об изображениях

Изображения взяты из открытых источников в интернете и используются **исключительно в ознакомительных и демонстрационных целях**. Все права на изображения принадлежат их законным владельцам. При коммерческом использовании необходимо заменить их на собственные или лицензионные материалы.

## Установка и запуск

1. Клонировать репозиторий
2. Добавить свои изображения в папку img/ (при необходимости)
3. Открыть index.html в любом современном браузере

## Требования

- Любой современный браузер (Chrome, Firefox, Safari, Edge)
